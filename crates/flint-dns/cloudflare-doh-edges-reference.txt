# Cloudflare DoH edge IPs — verified-working reference snapshot
#
# 100 Cloudflare anycast edge IPs confirmed serving `cloudflare-dns.com` DoH
# (HTTP/2, RFC 8484 / JSON `Status:0`) on 2026-06-24 from a US vantage.
#
# How: harvested A-records from ~4,200 top domains, filtered to Cloudflare's
# announced ranges, then probed each with SNI/Host = cloudflare-dns.com. A 206-IP
# sample verified 100% — any live, site-backed CF anycast IP answers the resolver,
# because Cloudflare runs DoH on the same global edge that fronts every CF site.
#
# Caveats:
#  - Anycast reachability is VANTAGE-DEPENDENT; a client elsewhere gets a different
#    (overlapping) working subset. The ranges are globally announced, the specific
#    live /32s vary by PoP catchment.
#  - IPs CHURN. This is a point-in-time snapshot, not a stable set.
#  - A censor blocks by RANGE, not by IP — so range coverage matters far more than
#    count. These 100 span only 5 ranges (top sites don't use CF's regional ranges).
#
# Use: a reference/candidate set for expanding flint-dns `default_pool()` or a future
# Ed25519-signed pool update. NOT auto-loaded by the crate (the baked-in pool embeds
# a small range-diverse subset; see pool.rs).
